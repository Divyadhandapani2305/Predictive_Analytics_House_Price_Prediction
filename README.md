# House Price Prediction - Predictive Analytics Task

## 📌 Project Overview
This project predicts house prices using the King County dataset. It was developed as part of the **Thiranex Predictive Analytics task**, which requires basic EDA, data cleaning, and comparing multiple machine learning models.

## 🛠️ Tech Stack
- **Language:** Python
- **IDE:** Visual Studio Code (VS Code)
- **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn

## 📊 Models Implemented & Compared
We trained and evaluated three different models to find the best predictor:
1. **Linear Regression** (Baseline Model)
2. **Decision Tree Regressor**
3. **Random Forest Regressor** (Ensemble Model)

## 📈 Results
- **Linear Regression R² Score:** 0.7110
- **Random Forest R² Score:** 0.8537 (Best Performance)

## 🎯 Conclusion
The **Random Forest Regressor** is the best performing model for this dataset with the highest accuracy score of **0.8537** and the lowest prediction errors.
